# Git練習プロジェクト

このフォルダーはGitとGitHubの練習用です。

## 📚 このプロジェクトで学べること

1. **Gitの基本操作**
   - リポジトリの初期化
   - ファイルの追加・コミット
   - ブランチの作成・切り替え
   - マージの練習

2. **GitHubの基本操作**
   - リモートリポジトリの作成
   - プッシュ・プル
   - プルリクエスト（PR）の作成
   - イシューの作成・管理

3. **実践的なワークフロー**
   - 機能ブランチでの開発
   - コードレビューの流れ
   - コンフリクトの解決

## 🚀 始め方

### 1. Gitリポジトリを初期化
```bash
cd Git練習
git init
```

### 2. 最初のコミット
```bash
git add .
git commit -m "初回コミット: 練習プロジェクトの作成"
```

### 3. GitHubにリポジトリを作成
- GitHubで新しいリポジトリを作成
- リモートを追加
```bash
git remote add origin https://github.com/あなたのユーザー名/リポジトリ名.git
```

### 4. プッシュ
```bash
git branch -M main
git push -u origin main
```

## 📝 練習課題

### 課題1: 基本的なコミット
1. `練習ファイル1.txt` を編集
2. 変更をコミット
3. コミット履歴を確認

### 課題2: ブランチの作成
1. `feature/新機能` ブランチを作成
2. ブランチでファイルを編集
3. メインブランチにマージ

### 課題3: GitHub連携
1. GitHubでリポジトリを作成
2. ローカルの変更をプッシュ
3. GitHub上でファイルを編集
4. プルしてローカルに反映
