## Gitコマンド一覧

### 基本操作
- `git init` - リポジトリを初期化
- `git status` - 状態を確認
- `git add <ファイル>` - ファイルをステージング
- `git commit -m "メッセージ"` - コミット
- `git log` - コミット履歴を表示

### ブランチ操作
- `git branch` - ブランチ一覧
- `git branch <名前>` - ブランチ作成
- `git checkout <ブランチ>` - ブランチ切り替え
- `git merge <ブランチ>` - ブランチをマージ

### リモート操作
- `git remote add origin <URL>` - リモート追加
- `git push` - プッシュ
- `git pull` - プル
- `git clone <URL>` - クローン
